Cat Class (Part 2)

Program Description

In this assignment you will create a basic Cat class.

Program Requirements

Your class shall have the following constructors:

Constuctor Description
Cat ( ) Constructs a cat with full thirst, health, energy, and happiness.
Cat (String name) Constructs a cat with full thirst, health, energy, and happiness and assigns it the given name.

Your class shall have the following private attributes:

Type Name Initial Value Description
String name "" The name of the cat.
int hunger 100 The hunger value of the cat (100 is full; 0 is hungry)
int thirst 100 The thirst value of the cat (100 is hydrated; 0 is thirsty)
int energy 100 The energy value of the cat (100 is energized; 0 is exhausted)
int happiness 100 The happiness value of the cat (100 is happy; 0 is sad)
int age 0 The age of the cat in days

Your Cat class shall have the following public methods:

Return Type Method Signature Description
void pet ( ) Adds 20 to the cats happiness. The cat outputs "Prr!"
void feed ( ) Adds 20 to the cats hunger.
void water ( ) Adds 20 to the cats thirst.
void play ( ) Adds 20 to the cats happiness. Subtract 20 from the cats energy.
void sleep ( ) Sets cats energy to 100. Resets everything else to 0. Increases cats age by 1 day.
void sleep (int hours) Increases cats energy by `hours`. Decreases everything else by `hours`. Does not effect cats age.
void setName (String name) Renames the cat.
String getName ( ) Returns the name of the cat.
int getHunger ( ) Returns the hunger value of the cat.
int getThirst ( ) Returns the thirst value of the cat.
int getHappiness ( ) Returns the happiness value of the cat.
int getEnergy ( ) Returns the energy value of the cat.
int getHeath ( ) Returns the health value of the cat. The health of the cat is determined by: health = (happiness + thirst + hunger + energy)/4 - (age/10). The health of the cat cannot be lower than 0.

Your Cat class shall have the following private methods:

Return Type Method Signature Description
void checkStatLimit ( ) Ensures the cats stats are within the range [0, 100]
